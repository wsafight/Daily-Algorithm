# 提升交互体验的 web Observer

web 领域有很多的 Observer，这些 Observer 可以实时反馈网页的某些交互变化。

Mutation Observer，可以观察 DOM 元素的增删以及属性变化。Resize Observer，可以观察元素的尺寸变化。而 Intersection Observer 是观察元素和窗体相交的状态。
